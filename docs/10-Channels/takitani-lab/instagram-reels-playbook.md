## Content Formula

1. First second: immediately recognizable modern location.
2. Main contrast: Joseon outfit, manners, or tools collide with the location.
3. Escalation: one unexpected action every 3-5 seconds.
4. Loop: ending should visually connect back to the opening or invite a rewatch.
5. CTA: ask viewers where to send the scholar next.

## Posting Time

Preferred KST windows:

- Tue/Wed/Thu: 19:30-20:30
- Friday: 17:30-18:30
- Sunday test slot: 12:00-14:00

## Caption Pattern

Use short Korean-first captions:

```text
조선 선비가 [장소]에 가면 벌어지는 일.

EP.XX - [LOCATION]
JOSEON UNIVERSE 계속됩니다.

다음 장소 추천은 댓글로.
```

Add 6-10 hashtags. Prefer specific tags over broad spam.

## Comment Prompt Pattern

```text
다음엔 어디 보낼까요?
1. PC방
2. 헬스장
3. 편의점
4. 지하철
```

## Do

- Keep scene concepts readable without sound.
- Build in a comment vote.
- Use clear location props: keyboard, headset, ramen, counter, subway gate, karaoke mic.
- Preserve music attribution when using CC BY assets.

## Avoid

- Tight close-ups that hide the location.
- Too many unrelated props in one cut.
- Separate per-scene BGM tracks that feel disconnected.
- Posting via CapCut share screen unless explicitly requested.
