# ADR: 캐릭터 시트 이미지 첨부 규칙 + 3인 가족 캐스트

## Context
EP02 이미지 1차 생성을 텍스트 DNA 앵커만으로 진행 → 컷 간 개체 드리프트 위험, 사용자 교정 2건 발생.

## Decision
1. 오늘묘/가족 캐릭터 등장 컷 생성 시 해당 **캐릭터 시트 PNG를 ChatGPT에 반드시 첨부**한다 (텍스트 앵커 단독 금지).
2. 세계관 캐스트를 **3인 가족**으로 확정: 아빠(주인공, 40대 중반 영포티) · 엄마(40대 중반 영포티) · 아들(12살 초5). 시트 = `character-sheet/family-*-sheet-v1.png`.
3. EP02 운반자 = 아빠 (컷1·2 얼굴 프레임 밖, 컷3 뒷모습).

## Consequences
- 시트 4종(오늘묘+가족 3)이 채널 자산으로 고정 — 新캐릭터 등장 시 시트부터 제작.
- `character_dna.md` 조연 섹션이 가족 스펙의 진실원천.
- ChatGPT 첨부는 클립보드 붙여넣기(cmd+V)로 자동화 (파일 업로드 도구는 로컬 경로 거부).
